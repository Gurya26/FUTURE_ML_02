# Support Ticket Classification using Machine Learning

## Project Overview

This project focuses on classifying customer support tickets into categories such as **Billing, Technical, and General** using Machine Learning.

It also assigns a **priority level (High, Medium, Low)** based on the issue described in the ticket.

---

## Technologies Used

* Python 🐍
* Pandas
* Scikit-learn
* TF-IDF Vectorizer
* Logistic Regression

---

## Features

* Text preprocessing using TF-IDF
* Ticket classification using Logistic Regression
* Priority prediction system
* Model evaluation using accuracy score
* Real-world simulation of support system

---

## Model Performance

* Accuracy: ~40%
* The model works correctly but can improve with more data.

---

## Sample Output

* Ticket: *"App is not working properly"*
* Predicted Category: **Technical**
* Priority: **Low**

---

## Insights

* Technical issues are most frequent
* Billing issues often require high priority
* Model performs better with larger datasets

---

## Conclusion

This project demonstrates how Machine Learning can be applied to automate customer support systems and improve response efficiency.
